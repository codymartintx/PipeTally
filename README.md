# PipeTally

PipeTally is an offline iPhone prototype for reading pipe tally sheet photos into editable joint lengths and exporting the result to Excel.

Current MVP:

- Select one or more tally sheet photos from the device.
- OCR each sheet into joint rows and measurements.
- Review each sheet independently, including the detected starting joint.
- Add more sheets later as the job continues.
- Export reviewed tally data to Excel with decimal and no-decimal length tabs.

The printable OCR-friendly tally sheet is saved in `Docs/PipeTallyOCRSheet.pdf`.
